# Reliable, Scalable and Maintainable applications:
---------------------------------------------------

1. Reliability:
   - Tolerating hardware and software faults
   - Human error
2. Scalability:
   - Measuring load and performance
   - Latency percentiles
   - Throughput
3. Maintainability:
   - Operability
   - Simplicity and evolvability
